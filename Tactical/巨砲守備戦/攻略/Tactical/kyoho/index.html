# 巨砲守備戦動画

砲台は何も強化していない状態である程度の得点を出す戦い方の参考資料になればと思い、試しに戦闘中の画面を録画してみました。
昨日は 7446 点を出していたのでそれよりは低いですが、参考になれば幸いです。

# 解説

後半はグダグダになってますが、現時点で私が心がけている倒し方は次の通りです。

  1. ゴキを最優先で倒す
    基本的に、ゴキを潰すことでボムのチャージが早く溜まるので、最優先でゴキを倒すと良いと思います。

  2. レーダーをチラチラ見ながら次のターゲットを決める

    • レーダーに映っている点は水色→黄色→赤の順で優先順位をつける
      優先度点の色敵の種類
      最水色ゴキ
      高黄色デカブツ
      低赤雑魚のゾンビなど

  3. デカブツはなるべくボムで倒す

  4. 斜めに走ってくる
    簡単にでしたが、以上です😊
