# 盟主ご挨拶

私は四代目盟主のエプロンです。
サーバ内の勢力争いが活発になっている中、Uee同盟と共に302サーバで生存を続けて参ります。
メンバーのプレーヤースキルと、レベルを協力しながら高めて行ければと思っております。

# BNA について

BNAはワールド#302で活動している同盟です。
正式名称はGANBARUNBAです。
基本的にPVEをメインに街を育てて行きたい方に特にオススメの同盟です。
お互い支えあっていきましょう♪
加入希望者大歓迎です！
詳細はゲーム内の当同盟の説明欄をご覧ください。

# 当同盟のポジション

- Uee同盟のアカデミーです。
- IPG同盟と連合です。

# 禁止事項
