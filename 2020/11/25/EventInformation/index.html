# 最強指揮官イベント

# 最強指揮官イベントとは？

# 目標の達成で稼いだポイントで競う

最強指揮官は、期間毎にイベントの目標が切り替わり、各目標を達成すると報酬が獲得できる。またそれとは別に稼いだポイントによって各段階ごとのランキング報酬、総合ランキング報酬が獲得できる。

# イベント目標は合計 5 種類

1～4 段階目のイベント目標はランダムな順番で 1 日ごとに切り替わり、5 段階目のみ固定で 3 日間開催される。

# イベント目標一覧

  1. 建物やテクノロジーのアップグレード量
  2. 資源の採集量
  3. ゾンビの討伐数
  4. 兵士の募集数
  5. 敵撃殺 (5 段階目)：撃破したユニット数

# 期間毎のイベントスケジュール

開催期間1 週間で 5 段階開催
1 段階目月曜 AM9:00～火曜 AM8:59
2 段階目火曜 AM9:00～水曜 AM8:59
3 段階目水曜 AM9:00～木曜 AM8:59
4 段階目木曜 AM9:00～金曜 AM8:59
5 段階目金曜 AM9:00～翌週月曜 AM8:59

# イベントの報酬

イベント全体を通して獲得した総合ポイントで順位が決まる。順位によって貰えるランキング報酬の数は増減する。

# 段階別ランキング報酬内容

# 一定ポイントで獲得できる報酬

段階ごとに得たポイントが一定に達したとき報酬が獲得できる。最後の報酬ではゴールドも手に入る。

  • ゴールド
  • 同盟名誉
  • 同盟ポイント

# 段階ごとのポイントの稼ぎ方

# 1～4 段階目はブーストを使用する

4 段階目まではブーストアイテム等を使用することで効率的にポイントを集められる。ハイスコアを狙うなら、イベント期間外は使わずに温存しておこう。

# 5 段階目は積極的に敵を撃破する

# 攻撃はどんどん仕掛けよう

5 段階目は敵ユニットを撃破した数でポイントが変動するため、勝てそうな敵にどんどん攻撃を仕掛けていこう。ただし相手が自分より強い場合や同盟に加入している場合は手痛い反撃を受ける可能性があるため、注意が必要となる

# 攻撃する際に同盟内で確認を行おう

同盟に加入している場合はむやみに攻撃すると同盟メンバーが敵に狙われたり、他同盟と戦闘が発生するといったデメリットがある。攻撃する際は目標となる敵ユニットを攻撃して良いか同盟内チャットで確認をしてから行おう。

# 敵が攻撃してきたら回避しよう

5 段階目では敵からの攻撃も激しくなる。敵が強いときは「平和の守護」を使用したり、兵士を同盟のフラッグや同盟要塞の共防に移動させてユニットの損失を防ごう。
