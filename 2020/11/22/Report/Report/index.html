Lv16 以下のメンバーのみ攻撃して行ったようです。

# 「(DOA) HyeonE」が我々の領土に入ってきて襲撃

戦力不足により協防できず


# 「(DOA)왕지」が「はゆゆゆさん」と「SUB さとけんさん」へ襲撃しようとしていたので

ログインしているメンバーで協防に入ったところ偵察に来た後
攻撃をやめて引き返していきました。
しかし、はゆさんは JPM に攻撃されて以来、ずっとログインしてないな。心配です。😭
