# 小技集

ここでは、同盟メンバーの皆さんが教えてもらったり、発見したりした便利な操作方法を共有して行きます。

# 同盟メンバーのログイン状態を一覧表示

同盟メンバーのログイン状態を一覧表示する方法のご紹介です。
何種類か方法はあるのですが、その中でも最も手軽に確認できる方法です。

  1. 通常のチャット欄に半角で「@」を入力します。
  2. すると、同盟メンバーが一覧になって表示されます。
  3. その時、オンラインか、何分前にログアウトしたかを確認することができます。

この時、「@」は削除しないと、「@」だけ発言することになって恥ずかしいので気を付けましょう（笑）
